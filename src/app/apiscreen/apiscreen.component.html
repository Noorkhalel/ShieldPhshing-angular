Authentication

To use the SafeSiteDetect API, secure your requests by authenticating with the provided API key. Include the API key in the header of your HTTP requests as demonstrated below:


import requests

url= "http://SafeSiteDetect.com/api/v1/Ai/Check" 
            
payload = {{ '{' }}
    "url": "https://www.example.com/", 
    "secretkey": "your_api_key"
{{ '}' }}
    
headers = {{ '{' }}
    "Content-Type": "application/json", 
{{ '}' }}
                
response = requests.post(url, json=payload, headers=headers)
    
if response.status_code == 200:
    json_data = response.json()
    print("API Response:", json_data)
else:
    print(f"Error: {{'{}'}}response.status_code{{'}'}}")
    print("Error Details:", response.text)
